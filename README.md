# Front-end-projects
This project is a simple Ola-themed webpage built using HTML and CSS. It displays scooters, bikes, service centers, and navigation links with images. The layout highlights various Ola models and features. All images are locally stored, and styling is handled through an external CSS file.

This project is a static webpage designed using HTML and CSS, inspired by the Ola Electric website. It showcases different Ola scooters, bikes, features, offers, and service center details using images and styled text. The goal is to practice front-end structure, layout, and design.

 Navigation bar with links

 Scooter portfolio section (Ola S1 series)

 Ola service center highlights

 Promotional offers and warranty details

 Multiple product images integrated

 Custom styling using an external index.css file
